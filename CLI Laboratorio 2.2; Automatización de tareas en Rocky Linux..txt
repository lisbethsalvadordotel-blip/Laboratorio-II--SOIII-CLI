Comandos utilizados, práctica laboratorio 2.2:
Automatización de tareas en Rocky Linux.

============================================

# Abrimos el archivo crontab para agregar tareas programadas:

sudo crontab -e

# Actualizar el sistema todos los días a las 11:00 p.m.

0 23 * * * /usr/bin/dnf update -y

# Reiniciar el sistema todos los domingos a las 3:00 a.m.

0 3 * * 0 /sbin/reboot

# Guardar y salir en vi:
# Presionar ESC, escribir :wq y presionar Enter

:wq

# Verificar las tareas programadas en cron:

sudo crontab -l

# Iniciar y habilitar el servicio atd:

sudo systemctl start atd
sudo systemctl enable atd

# Verificar que el servicio esté activo:

sudo systemctl status atd

# Mostrar el contenido de /tmp antes:

ls /tmp

# Programar la eliminación del contenido de /tmp dentro de 1 minuto:

echo "rm -rf /tmp/*" | at now + 1 minute

# Verificar la cola de tareas programadas con at:

atq

# Luego de 1 minuto, mostrar el contenido de /tmp después:

ls /tmp

# (Extra) Crear un archivo de prueba en /tmp dentro de 1 minuto:

echo "touch /tmp/PRUEBA.txt" | at now + 1 minute

# Verificar si el archivo fue creado:

ls /tmp
